# domya-fallen-link

Paper plugin for synchronizing Minecraft player data with the Domya website.

## Requirements

- Java 21
- Paper 1.21.x

## Build

```
./gradlew build
```

Output: `build/libs/domya-fallen-link-1.0.0.jar`

## Commands

- `/domyasync status`
- `/domyasync reload`
- `/domyasync sync`
- `/link <code>`
